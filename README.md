# Digital-Escape
Digital Escape, a  C++ console game built as an educational example for developers interested in learning fundamental programming concepts through interactive gameplay. The codebase is intentionally compact and avoids external dependencies, making it suitable for beginners.

The game's atmosphere is inspired by the classic science-fiction concept that there is no spoon and by the virtual world of matrix stories. Rather than recreating those settings directly, the project uses them as creative inspiration for a simple puzzle-based experience where players attempt to survive a sequence of digital challenges.

The source code demonstrates clean program organization with separate functions handling random events, scoring, and player interaction. The implementation favors readability over optimization so that learners can easily understand each programming construct before extending the project.
